# Lab 04 Report

## Scenario
A SIEM alert reported a possible malicious file upload to the company's web application.

## Investigation
- Captured HTTP traffic
- Located HTTP POST request
- Identified uploaded filename
- Verified HTTP response
- Collected source and destination IPs
- Followed HTTP Stream
- Documented indicators

## Findings
The upload request was successfully identified and analyzed.

## Severity
High

## Tier-1 Actions
- Validate alert
- Collect evidence
- Preserve PCAP
- Escalate to Tier-2
